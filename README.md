# ПРОБЛЕМЫ И ПЕРСПЕКТИВЫ ПРИМЕНЕНИЯ РОССИЙСКИХ НАБОРОВ ДАННЫХ ДЛЯ ОБУЧЕНИЯ ИСКУССТВЕННОГО ИНТЕЛЛЕКТА В МЕДИЦИНЕ

Код для моей статьи.

Файл `mean_image_count_per_pacient.py` предназначен для выводы статистики по количеству снимков на пациента для датасета ["Рассеянный склероз"](https://mosmed.ai/datasets/datasets/aie21selftestmri/).  
Файл `ANEURISM_dataset_stats.py` предназначен для вывода размерности файла `native_shuffled.nii` для датасета ["Аневризма аорты"](https://mosmed.ai/datasets/datasets/nabor-dannih-kompyuterno-tomograficheskoi-angiografii-s-priznakami-kaltsinoza-tromboza-dilatatsii-i-anevrizmi-i-soderzhaschii-segmentatsiyu-prosveta-i-stenki-bryushnogo-otdela-aorti/).  
Файл `head_neck_dataset_stats.py` предназначен для выводы статистики по количеству снимков на пациента для датасета ["КТ головы и шеи с оценкой лимфатических узлов"](https://mosmed.ai/datasets/datasets/mosmeddata-kt-golovi-i-shei-s-otsenkoi-limfaticheskih-uzlov-po-klassifikatsii-node-rads/).
